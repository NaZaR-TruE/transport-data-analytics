# Инженер-управленец нового типа: Аналитика в транспорте

## Профессиональное позиционирование
Я сочетаю фундаментальную инженерную школу СКФУ с 15-летним практическим опытом управления бизнесом в сфере пассажирских перевозок Ставропольского края. Мой подход — **Data-Driven Management** (управление на основе данных) в государственном и корпоративном секторах. Я не просто констатирую факты, а создаю алгоритмы, которые превращают разрозненные отчеты в четкую базу для принятия решений.

## Ключевые компетенции
*   **Отраслевой фундамент:** Знание технического устройства транспорта, логистики региона и законодательной базы (ФЗ-220, ФЗ-152).
*   **Аналитический стек:** Проектирование БД (PostgreSQL), написание сложных SQL-запросов, автоматизация отчетности на Python (Pandas).
*   **Управленческая ответственность:** Навык работы в режиме высокой ответственности за результат, финансовую дисциплину и безопасность.

## Решение прикладных задач ведомства
Мои навыки позволяют автоматизировать ключевые процессы аналитического департамента:
1.  **Консолидация данных:** Автоматический сбор и объединение «гор» таблиц Excel через SQL/Python, исключая ручной труд.
2.  **Контроль качества:** Создание скриптов для автоматической верификации отчетов перевозчиков и данных ГЛОНАСС (выявление аномалий и приписок).
3.  **Оптимизация сети:** Применение методов анализа данных для точного расчета потребности населения в маршрутах на основе цифр.

## Резюме
Считаю своей миссией внедрение современных инженерных и ИТ-решений в транспортную систему Ставропольского края. Обладаю уникальным для отрасли сочетанием практического опыта и компетенций в области Data Analysis. Готов к реализации проектов по цифровой трансформации департамента.
